# sysinfo
Command line loop which prints selected system variables to monitor

## Features
Monitor
- CPU Usage
- CPU Temp
- Memory Usage
- Load Average
- Uptime

## Usage:
1. Be on linux
2. Compile
3. Run

## Taster:
Menu:
            Select Variables to Monitor
            Press 's' to start!

               [ ] CPU Info
               [ ] CPU Temp
               [ ] Memory Info
               [ ] Load Avg. Info
               [ ] Uptime

Running:

            'q' to quit | 's' to change settings
